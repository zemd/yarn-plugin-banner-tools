# yarn-plugin-banner-tools

> Simple plugin that displays ascii banner

## Usage

```sh
yarn plugin import https://raw.githubusercontent.com/zemd/yarn-plugin-banner-tools/main/bundles/%40yarnpkg/plugin-banner-tools.js
```

and then

```sh
yarn banner Welcome!
```
